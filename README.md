# blog
PATOS Blog - made with Hugo!

## Desenvolvimento

### Clonar repo
```bash
git clone [repo url]
```

### Iniciar submodules
```bash
git submodule init
git submodule update
```

### Rodar Hugo
```bash
hugo server -D
```
